Tidsserier i pandas¶

Data med tidsinformation indeholder en rkke udfordringer ift. bde visualisering og databehandling.

  • Hvordan forstr Python tid og datoer?
  • Hvilken tidsenhed skal der arbejdes med?
  • Hvordan undgs for mange unikke vrdier i tidsvariablen?

I det flgende ses nrmere p, hvordan man kan arbejde med tidsinformation i Python. Som eksempeldata arbejdes med aktiekursdata for Maersk (A.P. Mller - Mrsk A/S).

Senere kombinerer vi dette med Google Trends data om sgninger p "Maersk".

Datast A: Aktiekurs for Maersk B

Datasttet "maerskb_price-history_01072021-01092021.csv" indeholder kursinformation for aktien "Maersk B". Data indeholder kursinformation for hver halve time for perioden 1/7/2021-1/9/2021; herunder kurs ved bning, luk, laveste kurs, hjeste kurs samt volumen.

Datasttet "google-trends_maersk_01072021-01092021.csv" indeholder antal sgninger p Google p "Maersk" i perioden 1/7/2021-1/9/2021.

Konvertering til datetime¶

Data indeholder tydeligvis tidsinformation i kolonnerne "Local Date" og "Local Time". Python kan bare ikke genkende, at det er dette format:

maersk['Local Time'] = maersk['Local Time'].astype('datetime64')

Håndtering af tidsdata¶

Pandas indeholder en lang rkke series metoder til at arbejde med datoer og tid ("datetime"). Disse er alle under .dt.

Man er i arbejdet med tidsserier ofte i situationer, hvor man kan have brug for at reducere detaljegraden. Fx er det vanskeligt i det nuvrende format at gennemskue hvilken mned, der blev handlet flest aktier. Samtidig kan en visualiseirng ogs med fordel simplificeres for at gre den rarere at se p.

Med de forskellige .dt metoder, kan man hurtigt danne nye kolonner baseret p en anden frekvens (fx timer, dage, uger, mneder, r). Data kan derefter grupperes og summeres efter disse kolonner.

Eksempler p .dt metoder:

  • dt.year: r
  • dt.month: Mned (i tal)
  • dt.day: Dag
  • dt.hour: Timetal
  • dt.date: Dato
  • dt.isocalendar().week: Ugenummer
  • dt.weekday: Ugedag (i tal)

Man kunne fx undersge, hvilken dag p ugen, der bliver handlet flest aktier, ved frst at lave en kolonne for ugedagen, gruppere p ugedag, danne middel for volumen og visualisere:

In [53]:
from plotnine import geom_bar

maersk['weekday'] = maersk['Local Time'].dt.weekday
maersk_weektrade = maersk.groupby('weekday')['Volume'].mean().to_frame(name = 'mean_volume').reset_index()

(ggplot(data = maersk_weektrade, mapping = aes(x = 'weekday', y = 'mean_volume'))
 + geom_bar(stat = 'identity')) # stat = identity fortller at data skal bruges, som det er

Man kunne ogs bruge metoderne til at simplificere grafen over lukkekurser en smule:

In [54]:
from plotnine import geom_line, geom_point, geom_text, scale_x_date

maersk['date'] = maersk['Local Time'].dt.date
maersk_weekly = maersk.loc[maersk.groupby('date')['Local Time'].idxmax(), :]
maersk_weekly = maersk_weekly.loc[maersk_weekly['weekday'] == 0]

(ggplot(data = maersk_weekly, mapping = aes(x = 'Local Time', y = 'Close'))
 + geom_line(color = "grey")
 + geom_point(size = 4, color = "black", fill = "lightblue")
 + geom_text(mapping = aes(label = 'Close'), nudge_y = 40)
 + scale_x_date(date_breaks = "1 week", labels = maersk_weekly['date'].unique()))
